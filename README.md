## Project Description
This project is a simple Pastry Shop system implemented using Java.
It demonstrates object-oriented programming concepts such as:
- Inheritance
- Polymorphism
- Encapsulation

The system includes different types of pastries (sweet and savory) and a pastry shop
that manages them. A basic GUI is also included to interact with the system.

## Project Structure

- SavoryPastry.java
- PastryShop.java
- PastryShopGUI.java
- Main.java
